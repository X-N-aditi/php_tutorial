variable
-----------
variable are containers to store data

rules for php variable name 
-------------------------------
1. a variable must starts with $ sign, followed by variable name
2. a variable name must be start with letter or underscore
3. a variable name can not be start with digit 
4. a variable name can conatin only alpha-numeric data like A-Z , a-z and 0-9
5. a vaiarble name are case sensitive
$age and $Age are different

 $GLOBALS - a array that contains refrence to all global variable of the scripting
 $_SERVER - hold web server configuration
 $_REQUEST - hold request for data get post 
 $_POST - receive HTTP POST request
 $_GET - receive HTTP GET request
 $_FILES - an array to upload items via http post method(file path , file type)
 $_ENV - Holds environment variables 
 $_COOKIE - An array of variables passed to the current script via HTTP Cookies
 $_SESSION - An array of session variables



 $_GET and $_POST are super globals , it means they can be accesible from anywhere or outside of scope
 $_GET are visible to others , so we can not use that in secure data , and also it has 2000 chars limit
 $_POST are not visibles to others , so we can use this in sensitive data like password and bank credential
